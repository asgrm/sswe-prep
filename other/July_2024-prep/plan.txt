process is a running program in the operating system. It has its own memory and cannot see nor access the memory of other running programs.

There are several ways to pass the data between two threads. By default this data is cloned i.e. both sender and receiver has their own copy of the data. And this data is cloned with ‘The Structured Cloned Algorithm’. If the data which is passed is deeply nesting within, then it’ll take a significant processing power to clone the data. So its not always optimal to have copies of data.
